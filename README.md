# Web Gomoku

纯静态网页版 AI 五子棋，支持：

- 响应式布局，适配桌面和手机
- 低级 / 中级 / 高级 AI
- 玩家先手或后手
- 悔棋
- 黑棋禁手检测开关
- 落子次序显示切换
- 玩家 AI 托管与托管等级选择
